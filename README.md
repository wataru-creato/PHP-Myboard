# PHP Study Project - 掲示板アプリ

## 📘 概要

PHPとMySQLを使ったシンプルな掲示板アプリです。
PHP学習でインプットが終わったため，アウトプットを含めてDBと連携した掲示板を制作しました。
それぞれユーザのログインログアウトができるようになっており、新規作成で新しいログインユーザやパスワードを登録できます。
掲示板では作成、読み取り、更新、削除のCRUDを意識して機能を追加しました。
---

## 🧩 主な機能

* 🔐 **ログイン機能**
  既存ユーザーがユーザ名とパスワードでログイン可能。

* 📝 **ユーザ登録**
  新しいユーザーを登録し、登録直後にそのままログイン可能。

* 🖊 **投稿の編集や削除**
  自分の投稿内容のみのコメントを編集、削除できる。

* 💬 **DBとの接続**
  ユーザやパスワード、掲示板の書き込んだユーザ名、投稿日、コメント内容をDB上で記録。
